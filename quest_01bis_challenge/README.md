# **Challenge**

## **Spring Blog !**

En t'appuyant sur les informations de cette quête, initialise ton propre projet `MyBlog` (tu peux passer l'étape "Ajout d'un Contrôleur REST").

Cette fois-ci, ajoute les dépendances suivantes :

- `Spring Web`
- `Spring Boot DevTools`
- `Spring Data JPA`
- `MySQL Driver`

_Le rôle de chacune de ces dépendances te sera expliqué au fur et à mesure de leur utilisation._

<aside>
<img src="/icons/info-alternate_blue.svg" alt="/icons/info-alternate_blue.svg" width="40px" />

Si tu essaies de démarrer ton serveur, le message suivant s'affichera :

`Failed to configure a DataSource: 'url' attribute is not specified and no embedded datasource could be configured.`
Pas d'inquiétude, c'est tout à fait normal ! Je t’invite à poursuivre avec la prochaine quête pour continuer la configuration de ton application.

</aside>

Versionne ton code sur *Github* et met à disposition le lien vers ton repository.

Tu l'auras compris, le but des prochaines quêtes va être de développer une API Spring pour un blog.

## **Critères de validation**

- Le dépôt *GitHub* contient bien le fichier `MyBlogApplication.java`
- Le fichier `pom.xml` contient les dépendances `spring-boot-starter-web`, `spring-boot-devtools`, `spring-boot-starter-data-jpa`, `mysql-connector-j`
